### Overview

This endpoint can be used to get a single shipment.

Endpoint: `/api/v1/shipments/{shipment_id}`

Method: `GET`

### Response

<ResponseField name="data" type="array" required>
    An array containing a single [Shipment](/guide/schemas/shipment) object.
</ResponseField>
