title: 'Delete Shipment'

### Overview

This endpoint can be used to delete a shipment.

Endpoint: `/api/v1/shipments/{shipment_id}`

Method: `DELETE`

### Response

If the shipment is successfully deleted, the API will return a `204 No Content` status code.
