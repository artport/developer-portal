---
title: 'Delete Shipment'
---

### Overview

This endpoint can be used to delete a shipment.

### Response

If the shipment is successfully deleted, the API will return a `204 No Content` status code.

<RequestExample>
    ```http Endpoint
    DELETE /api/v1/shipments/{shipment_id}
    ```
</RequestExample>

<ResponseExample>
    ```text Example Response Structure (Status Code: 204)
    No Content
    ```
</ResponseExample>
