### Overview

This endpoint can be used to get a single address.

### Response

<ResponseField name="data" type="array" required>
    An array containing a single [Address](/guide/schemas/address) object.
</ResponseField>

<RequestExample>

    ```http Endpoint
    GET /api/v1/addresses/{address_id}
    ```
</RequestExample>
