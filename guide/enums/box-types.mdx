### Overview

The following enum values represent the box types that are supported by the ArtPort API:

| ID | Box Type | Description                           |
|----|----------|---------------------------------------|
| 1  | Small    | 22in (L) x 18in (W) x 4in (H) - 45lbs |
| 2  | Medium   | 36in (L) x 24in (W) x 4in (H) - 45lbs |
| 3  | Large    | 44in (L) x 34in (W) x 4in (H) - 45lbs |
