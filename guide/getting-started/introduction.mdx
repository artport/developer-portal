### Overview

This developer portal contains the documentation required to assist software developers in working with the ArtPort API. Follow the steps below to get started building your integration.

<Steps>
    <Step title="Get an API token">
        Begin by visiting your organization's ArtPort account and [creating an API token](https://app.artport.co/organization/token) to authenticate with.
    </Step>
    <Step title="Review the documentation">
        Once you have your API token, you can [start building your own integration](/guide/api/shipments/create) by using the ArtPort APIs.
    </Step>
</Steps>

If you encounter a problem or issue while using the API, then please contact us via the ArtPort support center.
